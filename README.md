# Connected
**Your Intelligent Platform to learn how to connect with people and stay connected**

From awkward to awesome! Stop dreading conversations. Our AI-powered platform turns you into a confident communicator through micro-lessons, real practice and smart insights and rolling news briefs.

## Key Features
- **Micro-Learning Lessons**: Bite-sized educational content optimized for mobile consumption.
- **Rolling News Briefs**: AI-curated news summaries tailored to your interests.
- **Personalized Content**: Adaptive learning paths that adjust to your preferences and progress.
- **AI-Powered Curation**: Intelligent content selection using **LangGraph** multi-agent workflows.
- **Mobile-First Design**: Responsive UI built with modern web technologies for seamless mobile experience.
- **Interactive Learning**: Engaging lesson formats with progress tracking and analytics.
- **Smart Notifications**: Stay updated with personalized content recommendations.

## Tech Stack
**Frontend** (Deployed on Vercel/Cloudflare)
- **Next.js + React + TypeScript**
- **Material UI** (Custom Premium Theme)
- **Framer Motion** (Smooth Animations & Transitions)
- **VAPI Web SDK** (@vapi-ai/web) for voice interactions
- **Modern UI Components** (Custom Button, Card, AppShell)

**Backend** (Deployed on Cloud Platform)
- **FastAPI** (Python 3.11+)
- **Supabase**
- **AI Core**:
  - **LangGraph** multi-agent workflow management
  - **LLM**: **Claude**, **OpenAI** for content generation and personalization
  - **Voice AI**: **VAPI** for interactive voice practice sessions
